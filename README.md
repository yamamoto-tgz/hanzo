# hanzo

読み取り専用の Slack クライアントです。

## Install

```
$ git clone https://github.com/yamamoto-tgz/hanzo.git
$ cd hanzo

$ python3 -m venv venv
$ source ./venv/bin/activate

$ pip install -r requirements.txt
```

## Run

```
$ SLACK_USER_TOKEN=xxxxx flask run
```

## Environment Variables

-   SLACK_USER_TOKEN
